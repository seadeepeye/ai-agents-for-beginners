## рҹҢұ бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ

бһҳбһ»бһҒбһңбһ·бһҮбҹ’бһҮбһ¶бһ“бҹҒбҹҮбһҳбһ¶бһ“бһҳбҹҒбһҡбҹҖбһ“бһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһ—бҹҒбһ‘ AIбҹ” бһҳбҹҒбһҡбҹҖбһ“бһ“бһёбһҳбһҪбһҷбҹ—бһӮбһәбһҳбһ¶бһ“бһ”бҹ’бһҡбһ’бһ¶бһ“бһ”бһ‘бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ–бһёбһҖбһ“бҹ’бһӣбҹӮбһ„бһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸ!

бһҳбһ¶бһ“бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ—бһ¶бһҹбһ¶бһ…бҹ’бһҡбһҫбһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһ»бһҒбһңбһ·бһҮбҹ’бһҮбһ¶бһ“бҹҒбҹҮбҹ” бһ…бһјбһӣбһ‘бҹ…бһҖбһ¶бһ“бҹӢ [бһ—бһ¶бһҹбһ¶бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһ“бҹ…бһ‘бһёбһ“бҹҒбҹҮ](#-multi-language-support)бҹ”

бһ”бһҫбһ“бҹҒбҹҮбһҮбһ¶бһӣбһҫбһҖбһҠбҹҶбһ”бһјбһ„бһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҠбҹӮбһӣ Generative AI бһҹбһјбһҳбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҳбһ»бһҒбһңбһ·бһҮбҹ’бһҮбһ¶ [Generative AI For Beginners](https://aka.ms/genai-beginners) бһҡбһ”бһҹбҹӢбһҷбһҫбһ„ бһҠбҹӮбһӣбһҳбһ¶бһ“бһҳбҹҒбһҡбҹҖбһ“ 21 бһӣбһҫбһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶бһҳбһҪбһҷ GenAIбҹ”

бһҖбһ»бҹҶбһ—бҹ’бһӣбҹҒбһ… [бһ•бҹ’бһҖбһ¶бһҷ (рҹҢҹ) бһҡбҹүбҹҒбһ бҹ’бһҹбҹ’бһ–бһјбһ“бҹҒбҹҮ](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) бһ“бһ·бһ„ [бһ…бһҳбҹ’бһӣбҹӮбһҖбһҡбҹүбҹҒбһ бҹ’бһҹбҹ’бһ–бһјбһ“бҹҒбҹҮ](https://github.com/microsoft/ai-agents-for-beginners/fork) бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҫбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһјбһҠбҹ”

## бһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳ

бһӮбһҳбҹ’бһҡбҹ„бһ„бһ“бҹҒбҹҮбһҹбҹ’бһңбһ¶бһӮбһҳбһ“бҹҚбһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳбһ“бһ·бһ„бһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһҷбҹ„бһ”бһӣбҹӢбҹ” бһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳбһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҠбһ¶бһҖбҹӢбһҹбһјбһҳбһҹбҹ’бһҡбһ”бһҸбһ¶бһҳ
бһҹбһ“бҹ’бһ’бһ·бһҹбһүбҹ’бһүбһ¶бһҹбһ·бһ‘бҹ’бһ’бһ·бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ…бһјбһӣбһҡбһҪбһҳ (CLA) бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһўбҹ’бһ“бһҖбһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’бһ· бһ“бһ·бһ„бһ–бһ·бһҸбһ”бҹ’бһҡбһ¶бһҖбһҠбһ•бҹ’бһҸбһӣбҹӢбһҹбһ·бһ‘бҹ’бһ’бһ·
бһҠбһӣбҹӢбһҷбһҫбһ„бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһӣбһҳбҹ’бһўбһ·бһҸ бһҹбһјбһҳбһ‘бҹ…бһҖбһ¶бһ“бҹӢ <https://cla.opensource.microsoft.com>бҹ”

бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһҠбһ¶бһҖбҹӢбһҹбҹҶбһҺбһҫ pull request бһҳбһҪбһҷ бһ”бһ»бһҸ CLA бһ“бһ№бһ„бһҖбҹҶбһҺбһҸбҹӢбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҗбһ¶бһҸбһҫбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһң

бһҠбҹӮбһӣбһ”бһ»бһҸбһ•бҹ’бһҸбһӣбҹӢбҹ” бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ’бҹ’бһңбһҫбһҸбҹӮбһҳбҹ’бһҸбһ„бһҸбҹӮбһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбҹҒбһ бҹ’бһңбһ”бһ¶бһ‘бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ”бҹ’бһҡбһҫ CLA бһҡбһ”бһҹбҹӢбһҷбһҫбһ„бҹ”
