## 🌱 ਸ਼ੁਰੂਆਤ ਕਰਨਾ

ਇਸ ਕੋਰਸ ਵਿੱਚ AI ਏਜੰਟ ਬਣਾਉਣ ਦੇ ਮੂਲਾਂਕਣ ਨੂੰ ਕਵਰ ਕੀਤਾ ਗਿਆ ਹੈ। ਹਰ ਪਾਠ ਆਪਣਾ ਵਿਸ਼ਾ ਸਿੱਖਾਉਂਦਾ ਹੈ ਇਸ ਲਈ ਜਿੱਥੇ ਚਾਹੋ ਉੱਥੇ ਸ਼ੁਰੂ ਕਰੋ!

ਇਸ ਕੋਰਸ ਲਈ ਬਹੁ-ਭਾਸ਼ਾ ਸਹਾਇਤਾ ਹੈ। ਸਾਡੀਆਂ [ਉਪਲਬਧ ਭਾਸ਼ਾਵਾਂ ਇੱਥੇ](#-multi-language-support) ਚੈੱਕ ਕਰੋ। 

ਜੇ ਇਹ ਤੁਹਾਡਾ ਪਹਿਲਾ ਵਾਰ ਜਨਰੇਟਿਵ AI ਮਾਡਲ ਨਾਲ ਬਣਾਉਣਾ ਹੈ, ਤਾਂ ਸਾਡੇ [ਸ਼ੁਰੂਆਤੀ ਲਈ ਜਨਰੇਟਿਵ AI](https://aka.ms/genai-beginners) ਕੋਰਸ ਨੂੰ ਵੇਖੋ, ਜਿਸ ਵਿੱਚ GenAI ਨਾਲ ਬਣਾਉਣ ਦੇ 21 ਪਾਠ ਸ਼ਾਮਿਲ ਹਨ।

ਇਸ ਰੀਪੋ ਨੂੰ [ਸਟਾਰ (🌟) ਕਰਨਾ](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ਅਤੇ ਕੋਡ ਚਲਾਉਣ ਲਈ [ਫੋਰਕ ਕਰਨਾ](https://github.com/microsoft/ai-agents-for-beginners/fork) ਨਾ ਭੁੱਲੋ।

### ਹੋਰ ਲੇਰਨਰਾਂ ਨਾਲ ਮਿਲੋ, ਆਪਣੇ ਸਵਾਲਾਂ ਦੇ ਜਵਾਬ ਲਓ

ਜੇ ਤੁਸੀਂ ਫਸ ਜਾਂਦੇ ਹੋ ਜਾਂ AI ਏਜੰਟ ਬਣਾਉਣ ਬਾਰੇ ਕੋਈ ਵੀ ਸਵਾਲ ਹੈ, ਤਾਂ ਸਾਡੀ ਸਮਰਪਿਤ ਡਿਸਕੋਰਡ ਚੈਨਲ ਵਿੱਚ ਸ਼ਾਮਿਲ ਹੋਵੋ [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) ਤੇ।

### ਤੁਹਾਨੂੰ ਕੀ ਚਾਹੀਦਾ ਹੈ

ਇਸ ਕੋਰਸ ਦਾ ਹਰ ਪਾਠ ਕੋਡ ਉਦਾਹਰਨਾਂ ਨਾਲ ਹੈ, ਜੋ code_samples ਫੋਲਡਰ ਵਿੱਚ ਮਿਲਣਗੀਆਂ। ਤੁਸੀਂ [ਫੋਰਕ ਕਰ ਕੇ](https://github.com/microsoft/ai-agents-for-beginners/fork) ਆਪਣੀ ਕਾਪੀ ਬਣਾ ਸਕਦੇ ਹੋ।  

ਇਹ ਪ੍ਰੈਕਟਿਸ ਮਾਈਕ੍ਰੋਸੌਫਟ ਏਜੰਟ ਫਰੇਮਵਰਕ ਨਾਲ ਐਜ਼ੂਰੇ AI Foundry Agent Service V2 ਦੀ ਵਰਤੋਂ ਕਰਦੀਆਂ ਹਨ:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure ਖਾਤਾ ਲੋੜੀਂਦਾ ਹੈ

ਇਹ ਕੋਰਸ ਹੇਠਾਂ ਦਿੱਤੇ Microsoft ਦੇ AI ਏਜੰਟ ਫਰੇਮਵਰਕ ਅਤੇ ਸੇਵਾਵਾਂ ਦੀ ਵਰਤੋਂ ਕਰਦਾ ਹੈ:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


ਇਸ ਕੋਰਸ ਲਈ ਕੋਡ ਚਲਾਉਣ ਬਾਰੇ ਹੋਰ ਜਾਣਕਾਰੀ ਲਈ, [ਕੋਰਸ ਸੈਟਅੱਪ](./00-course-setup/README.md) ਵਿੱਚ ਜਾਓ।
