## 🌱 Začínáme

Tento kurz obsahuje lekce pokrývající základy tvorby AI agentů. Každá lekce se věnuje svému tématu, takže můžete začít kdekoli chcete!

Kurz podporuje vícejazyčnost. Podívejte se na naše [dostupné jazyky zde](#-multi-language-support).

Pokud je to poprvé, co pracujete s modely Generative AI, podívejte se na náš kurz [Generative AI For Beginners](https://aka.ms/genai-beginners), který obsahuje 21 lekcí o tvorbě s GenAI.

Nezapomeňte [ohodnotit (🌟) tento repozitář](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) a [forknout tento repozitář](https://github.com/microsoft/ai-agents-for-beginners/fork) pro spuštění kódu.

### Seznamte se s dalšími studenty, získejte odpovědi na své otázky

Pokud uvíznete nebo máte jakékoli otázky ohledně tvorby AI agentů, připojte se k našemu speciálnímu kanálu Discord v [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Co potřebujete

Každá lekce v tomto kurzu obsahuje příklady kódu, které najdete ve složce code_samples. Můžete si [forknout tento repozitář](https://github.com/microsoft/ai-agents-for-beginners/fork) a vytvořit si vlastní kopii.

Příklady kódu v těchto cvičeních využívají Microsoft Agent Framework s Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Vyžaduje účet Azure

Tento kurz využívá následující AI agentní frameworky a služby od Microsoftu:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


Pro více informací o spuštění kódu pro tento kurz přejděte na [Nastavení kurzu](./00-course-setup/README.md).
