## 🌱 Початок роботи

Цей курс містить уроки, що охоплюють основи створення AI Агентів. Кожен урок охоплює свою тему, тому починайте з будь-якого, що вам подобається!

Для цього курсу є підтримка кількох мов. Перейдіть до нашого розділу [доступних мов тут](#-multi-language-support). 

Якщо ви вперше працюєте з генеративними моделями ШІ, перегляньте наш курс [Генеративний ШІ для Початківців](https://aka.ms/genai-beginners), який включає 21 урок зі створення на базі GenAI.

Не забудьте [поставити зірочку (🌟) цьому репозиторію](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) та [форкнути цей репозиторій](https://github.com/microsoft/ai-agents-for-beginners/fork), щоб запускати код.

### Познайомтесь з іншими учнями, отримаєте відповіді на запитання

Якщо у вас виникнуть труднощі або запитання щодо створення AI Агентів, приєднуйтесь до нашого спеціального Discord-каналу у [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Що вам потрібно

Кожен урок цього курсу містить приклади коду, які можна знайти у папці code_samples. Ви можете [форкнути цей репозиторій](https://github.com/microsoft/ai-agents-for-beginners/fork), щоб створити власну копію.

Приклади коду у цих вправах використовують Microsoft Agent Framework з Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Потрібен обліковий запис Azure

У цьому курсі використовуються такі фреймворки і сервіси AI Агентів від Microsoft:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


Для більш детальної інформації про запуск коду цього курсу перейдіть до [Налаштування курсу](./00-course-setup/README.md).
