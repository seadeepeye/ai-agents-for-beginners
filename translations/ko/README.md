## 🌱 시작하기

이 강의는 AI 에이전트 구축의 기본을 다룹니다. 각 강의는 자체 주제를 다루므로 원하는 곳부터 시작하세요!

이 강의는 다국어 지원을 제공합니다. [여기서 지원 언어를 확인하세요](#-multi-language-support). 

생성 AI 모델을 처음 사용한다면, 21개의 강의로 구성된 [초보자를 위한 생성 AI](https://aka.ms/genai-beginners) 강의를 확인해 보세요.

이 저장소에 [별(🌟)을 눌러주세요](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) 그리고 코드를 실행하려면 [저장소를 포크하세요](https://github.com/microsoft/ai-agents-for-beginners/fork).

### 다른 학습자 만나기, 질문 해결하기

AI 에이전트 구축 중 궁금한 점이 있거나 어려움이 생기면 [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord)의 전용 디스코드 채널에 참여하세요.

### 필요한 것

각 강의는 code_samples 폴더에서 찾을 수 있는 코드 예제를 포함합니다. [이 저장소를 포크](https://github.com/microsoft/ai-agents-for-beginners/fork)하여 자신의 복사본을 생성할 수 있습니다.  

이 연습의 코드 예제는 Microsoft Agent Framework와 Azure AI Foundry Agent Service V2를 활용합니다:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure 계정 필요

이 강의에서는 Microsoft의 다음 AI 에이전트 프레임워크와 서비스를 사용합니다:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


이 강의의 코드 실행에 대한 자세한 정보는 [강의 설정](./00-course-setup/README.md)에서 확인하세요.
