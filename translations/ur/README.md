## 🌱 شروع کریں

یہ کورس AI ایجنس بنانے کے بنیادی اصولوں پر اسباق فراہم کرتا ہے۔ ہر سبق اپنا موضوع کور کرتا ہے، تو جہاں چاہیں وہاں سے شروع کریں!

اس کورس کے لیے کثیراللسانی سپورٹ موجود ہے۔ ہماری [دستیاب زبانیں یہاں دیکھیں](#-multi-language-support)۔

اگر آپ پہلی بار جنریٹو AI ماڈلز کے ساتھ کام کر رہے ہیں، تو ہمارا [جنریٹو AI برائے مبتدئین](https://aka.ms/genai-beginners) کورس دیکھیں، جس میں GenAI کے ساتھ بنانے کے 21 اسباق شامل ہیں۔

اس ریپو کو [سٹار (🌟) کرنا](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) اور [فورک کرنا](https://github.com/microsoft/ai-agents-for-beginners/fork) نہ بھولیں تاکہ آپ کوڈ چلا سکیں۔

### دوسرے سیکھنے والوں سے ملیں، اپنے سوالات کے جواب پائیں

اگر آپ پھنس جائیں یا AI ایجنس بنانے کے بارے میں کوئی سوال ہو تو ہمارے مخصوص Discord چینل میں شامل ہوں [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord)۔

### آپ کو کیا چاہیے

اس کورس کا ہر سبق کوڈ کے مثالوں کے ساتھ ہے، جو کوڈ_سمپلز فولڈر میں مل سکتے ہیں۔ آپ اپنی کاپی بنانے کے لیے [اس ریپو کو فورک کریں](https://github.com/microsoft/ai-agents-for-beginners/fork)۔

ان مشقوں کے کوڈ مثالیں مائیکروسافٹ ایجنٹ فریم ورک کے ساتھ Azure AI Foundry Agent Service V2 استعمال کرتی ہیں:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure اکاؤنٹ ضروری

یہ کورس مائیکروسافٹ کے مندرجہ ذیل AI ایجنٹ فریم ورکس اور سروسز استعمال کرتا ہے:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


اس کورس کے کوڈ کو چلانے کی مزید معلومات کے لیے جائیں [کورس سیٹ اپ](./00-course-setup/README.md)۔
