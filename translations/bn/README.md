>
> এই রিপোজিটরিতে ৫০+ ভাষার অনুবাদ অন্তর্ভুক্ত যা ডাউনলোড সাইজ উল্লেখযোগ্যভাবে বৃদ্ধি করে। অনুবাদ ছাড়া ক্লোন করতে, sparse checkout ব্যবহার করুন:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> এটি আপনাকে দ্রুত ডাউনলোড সহ কোর্স সম্পূর্ণ করার জন্য প্রয়োজন সবকিছু দেবে।

## 🌱 শুরু করা

এই কোর্সে AI এজেন্ট তৈরি করার মৌলিক বিষয়বস্তু কভার করা হয়েছে। প্রতিটি পাঠ নিজের আলাদা বিষয় নিয়ে কাজ করে, তাই যেখান থেকেই শুরু করতে পারেন!

এই কোর্সের জন্য বহুভাষা সমর্থন রয়েছে। আমাদের [উপলভ্য ভাষাসমূহ এখানে](#-multi-language-support) দেখুন।

যদি এটি আপনার প্রথমবার জেনেরেটিভ AI মডেল দিয়ে তৈরি করা হয়, আমাদের [Generative AI For Beginners](https://aka.ms/genai-beginners) কোর্স দেখুন, যেখানে জেনAI দিয়ে নির্মাণের ২১টি পাঠ আছে।

ভুলবেন না [এই রিপোতে (🌟) তারা দিন](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) এবং [রিপোটি fork করুন](https://github.com/microsoft/ai-agents-for-beginners/fork) কোড চালানোর জন্য।

### অন্য শিক্ষার্থীদের সাথে যুক্ত হোন, আপনার প্রশ্নের উত্তর পান

যদি আটকে পড়েন বা AI এজেন্ট তৈরির ব্যাপারে কোনো প্রশ্ন থাকে, আমাদের নিবেদিত Discord চ্যানেলে যোগ দিন [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) এ।

### যা আপনার প্রয়োজন

এই কোর্সের প্রতিটি পাঠে কোড উদাহরণ রয়েছে, যা code_samples ফোল্ডারে পাওয়া যাবে। আপনি [এই রিপোটি fork করতে পারেন](https://github.com/microsoft/ai-agents-for-beginners/fork) নিজের কপি তৈরির জন্য।

এই অনুশীলনের কোড উদাহরণগুলো Microsoft Agent Framework এর সাথে Azure AI Foundry Agent Service V2 ব্যবহার করে:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure অ্যাকাউন্ট প্রয়োজন

এই কোর্সে Microsoft-এর নিম্নলিখিত AI Agent ফ্রেমওয়ার্ক ও সার্ভিস ব্যবহার করা হয়েছে:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


এই কোর্সের কোড চালানোর আরো তথ্যের জন্য যান [Course Setup](./00-course-setup/README.md) এ।
