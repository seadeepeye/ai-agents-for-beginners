## 🌱 Kezdés

Ez a tanfolyam leckéket tartalmaz az AI ügynökök fejlesztésének alapjairól. Minden lecke a saját témáját fedi le, szóval kezdj bárhol, ahol szeretnéd!

A tanfolyam többnyelvű támogatást kínál. A [elérhető nyelveinkhez ide kattints](#-multi-language-support). 

Ha először használsz generatív AI modelleket, nézd meg a [Generatív AI Kezdőknek](https://aka.ms/genai-beginners) tanfolyamunkat, amely 21 leckét tartalmaz a GenAI fejlesztésről.

Ne felejtsd el [megcsillagozni (🌟) ezt a repo-t](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) és [forkolni ezt a repo-t](https://github.com/microsoft/ai-agents-for-beginners/fork), hogy futtatni tudd a kódot.

### Ismerkedj meg más tanulókkal, válaszokat kapj kérdéseidre

Ha elakadsz, vagy kérdésed van az AI ügynökök fejlesztésével kapcsolatban, csatlakozz dedikált Discord csatornánkhoz a [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) szerveren.

### Amit szükséges tudni

Minden lecke kódmintákat tartalmaz, melyek a code_samples mappában találhatók. [Forkold ezt a repo-t](https://github.com/microsoft/ai-agents-for-beginners/fork), hogy saját példányt készíts.

Ezek a kódminták a Microsoft Agent Frameworköt használják az Azure AI Foundry Agent Service V2-vel:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure fiók szükséges

Ez a tanfolyam a Microsoft alábbi AI ügynök keretrendszereit és szolgáltatásait használja:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


A tanfolyam kódjának futtatásáról további információkat a [Course Setup](./00-course-setup/README.md) alatt találsz.
