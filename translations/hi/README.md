> **स्थानीय रूप से क्लोन करना पसंद करते हैं?**
>
> इस रिपॉजिटरी में 50+ भाषाओं के अनुवाद शामिल हैं जो डाउनलोड का आकार काफी बढ़ा देते हैं। अनुवाद के बिना क्लोन करने के लिए, sparse checkout का उपयोग करें:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> यह आपको कोर्स पूरा करने के लिए आवश्यक सभी चीजें तेज डाउनलोड के साथ देता है।

## 🌱 शुरूआत करना

इस कोर्स में AI एजेंट्स बनाने के मूल सिद्धांतों को कवर करने वाले पाठ हैं। प्रत्येक पाठ अपना विषय कवर करता है इसलिए आप कहीं से भी शुरू कर सकते हैं!

इस कोर्स के लिए बहुभाषी समर्थन है। हमारे [उपलब्ध भाषाओं को यहाँ देखें](#-multi-language-support)। 

यदि यह आपका पहली बार जनरेटिव AI मॉडल्स के साथ निर्माण करने का अनुभव है, तो हमारा [Generative AI For Beginners](https://aka.ms/genai-beginners) कोर्स देखें, जिसमें GenAI के साथ निर्माण पर 21 पाठ शामिल हैं।

इस रिपॉजिटरी को [सितारा (🌟) देना न भूलें](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) और कोड चलाने के लिए इसे [फोर्क करें](https://github.com/microsoft/ai-agents-for-beginners/fork)।

### अन्य शिक्षार्थियों से मिलें, अपने प्रश्नों के उत्तर पाएं

यदि आप फंस जाते हैं या AI एजेंट्स निर्माण के बारे में कोई प्रश्न है, तो हमारे समर्पित Discord चैनल में शामिल हों [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) में।

### आपकी ज़रूरत क्या है

इस कोर्स के प्रत्येक पाठ में कोड उदाहरण शामिल हैं, जो code_samples फोल्डर में पाए जा सकते हैं। आप अपनी कॉपी बनाने के लिए इसे [फोर्क कर सकते हैं](https://github.com/microsoft/ai-agents-for-beginners/fork)।  

इन अभ्यासों में सार्वजनिक कोड उदाहरण Microsoft Agent Framework के साथ Azure AI Foundry Agent Service V2 का उपयोग करते हैं:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure खाता आवश्यक

यह कोर्स Microsoft से निम्नलिखित AI एजेंट फ्रेमवर्क और सेवाओं का उपयोग करता है:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


इस कोर्स के लिए कोड चलाने के बारे में अधिक जानकारी के लिए, [Course Setup](./00-course-setup/README.md) देखें।
