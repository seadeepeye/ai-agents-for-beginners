>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> Ово вам даје све што вам треба да завршите курс са много бржим преузимањем.

## 🌱 Почетак

Овај курс има лекције које покривају основе изградње AI агената. Свака лекција покрива своју тему тако да можете почети где год желите!

Овај курс има подршку за више језика. Погледајте наше [доступне језике овде](#-multi-language-support). 

Ако вам је ово први пут да радите са генеративним AI моделима, погледајте наш курс [Генеративни AI за почетнике](https://aka.ms/genai-beginners), који укључује 21 лекцију о раду са GenAI.

Не заборавите да [означите (🌟) ово складиште](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) и [форкујете ово складиште](https://github.com/microsoft/ai-agents-for-beginners/fork) како бисте могли да покренете код.

### Упознајте друге учеснике, добијте одговоре на питања

Ако застакнете или имате питања о прављењу AI агената, придружите се нашем посвећеном Discord каналу у [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Шта вам је потребно

Свака лекција у овом курсу садржи примере кода који се налазе у фасцикли code_samples. Можете [форковати ово складиште](https://github.com/microsoft/ai-agents-for-beginners/fork) да направите своју копију.  

Примери кода у овим вежбама користе Microsoft Agent Framework са Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Потребан Azure налог

Овај курс користи следеће AI Agent оквире и сервисе из Microsoft-а:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


За више информација о покретању кода за овај курс, идите на [Course Setup](./00-course-setup/README.md).
