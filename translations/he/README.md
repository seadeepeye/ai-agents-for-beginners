> **מעדיף לשכפל מקומית?**
>
> מאגר זה כולל יותר מ-50 תרגומים בשפות שמגדילים משמעותית את גודל ההורדה. לשכפול ללא תרגומים, השתמש ב-sparse checkout:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> זה נותן לך את כל מה שצריך כדי לסיים את הקורס עם הורדה מהירה יותר.

## 🌱 להתחיל

בקורס זה יש שיעורים המכסים את היסודות של בניית סוכני AI. כל שיעור עוסק בנושא משלו, אז התחילו איפה שאתם רוצים!

יש תמיכה מרובת שפות לקורס זה. גשו ל-[השפות הזמינות כאן](#-multi-language-support).

אם זו הפעם הראשונה שאתם בונים עם דגמי AI מחוללים, בדקו את הקורס שלנו [AI מחולל למתחילים](https://aka.ms/genai-beginners), הכולל 21 שיעורים על בנייה עם GenAI.

אל תשכחו [לסמן בכוכב (🌟) את מאגר זה](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ו-[לעשות Fork למאגר זה](https://github.com/microsoft/ai-agents-for-beginners/fork) כדי להריץ את הקוד.

### פגשו לומדים אחרים, קבלו מענה לשאלותיכם

אם נתקעתם או יש לכם שאלות בנוגע לבניית סוכני AI, הצטרפו לערוץ דיסקורד ייעודי שלנו ב-[Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### מה אתם צריכים

כל שיעור בקורס זה כולל דוגמאות קוד, שניתן למצוא בתיקיית code_samples. תוכלו [לעשות Fork למאגר זה](https://github.com/microsoft/ai-agents-for-beginners/fork) וליצור עותק משלכם.

דוגמאות הקוד בתרגילים האלה משתמשות במסגרת Microsoft Agent Framework עם שירות Azure AI Foundry Agent V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - דרושה חשבון Azure

קורס זה משתמש במסגרת סוכני AI ושירותים הבאים מ-Microsoft:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [שירות Azure AI Foundry Agent V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


למידע נוסף על הרצת הקוד לקורס זה, גשו ל-[הגדרת הקורס](./00-course-setup/README.md).
