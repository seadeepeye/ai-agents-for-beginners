> **స్థానికంగా క్లోన్ చేయాలనుకుంటున్నారా?**
>
> ఈ రిఫోజిటరీ 50+ భాషా అనువాదాలను కలిగి ఉంది, ఇది డౌన్లోడ్ పరిమాణాన్ని గణనీయంగా పెంచుతుంది. అనువాదాలు లేకుండా క్లోన్ చేయడానికి sparse checkout ను ఉపయోగించండి:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> ఇది కోర్సును పూర్తిచేయడానికి అవసరమైన ప్రతిదాన్ని వేగంగా డౌన్లోడ్ చేసేందుకు మీకు సహాయపడుతుంది.

## 🌱 ప్రారంభం చేయడం

ఈ కోర్సు AI ఏజెంట్లు నిర్మాణంలో పునాది అంశాలను అందించే పాఠాలు కలిగిఉన్నది. ప్రతి పాఠం దాని స్వంత విషయం కవర్ చేస్తుంది కాబట్టి మీరు ఇచ్ఛిన చోటుంచీ ప్రారంభించండి!

ఈ కోర్సుకు బహుభాషా మద్దతు ఉంది. మన [లభ్యమయ్యే భాషలు ఇక్కడ](#-multi-language-support) చూడండి.

మీరు జనరేటివ్ AI మోడల్స్‌తో మొదటిసారి నిర్మిస్తున్నట్లయితే, మా [ప్రారంభీకుల కోసం జనరేటివ్ AI](https://aka.ms/genai-beginners) కోర్సును చూడండి, ఇందులో GenAI తో నిర్మాణంపై 21 పాఠాలు ఉన్నాయి.

ఈ రిపోను [నక్షత్రం (🌟) వేయడం](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) మరువవద్దు, మరియు కోడ్ నడపడానికి ఈ రిపోను [fork చేయండి](https://github.com/microsoft/ai-agents-for-beginners/fork).

### ఇతర నేర్చుకునేవారిని కలవండి, మీ ప్రశ్నలకు జవాబు పొందండి

మీకు AI ఏజెంట్లు నిర్మాణంలో సమస్యలు వచ్చినట్లయితే లేదా ఎలాంటి ప్రశ్నలు ఉంటే, మాతో ప్రత్యేకంగా ఉన్న Discord ఛానెల్ [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) లో చేరండి.

### మీకు ఏమి అవసరం

ఈ కోర్సులో ప్రతి పాఠంలో కోడ్ ఉదాహరణలు ఉంటాయి, అవి code_samples ఫోల్డర్‌లో లభ్యమవుతాయి. మీరు మీ స్వంత కాపీ సృష్టించడానికి ఈ రిపోను [fork చేసుకోవచ్చు](https://github.com/microsoft/ai-agents-for-beginners/fork).

ఈ వ్యాయామాలలోని కోడ్ ఉదాహరణలు Microsoft Agent Framework ను Azure AI Foundry Agent Service V2 తో ఉపయోగిస్తాయి:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure ఖాతా అవసరం

ఈ కోర్సు Microsoft నుండి ఈ క్రింది AI ఏజెంట్ ఫ్రేమ్‌వర్క్‌లు మరియు సేవలను ఉపయోగిస్తుంది:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

ఈ కోర్సు కోసం కోడ్ ఎలా నడిపించాలో తెలుసుకోవడానికి [Course Setup](./00-course-setup/README.md) చూడండి.

## సహకారం

ఈ ప్రాజెక్ట్ కాంట్రిబ్యూషన్‌లు మరియు సూచనలను స్వాగతిస్తుంది.  చాలా కాంట్రిబ్యూషన్‌లు మీరు సంప్రదించే కంట్రిబ్యూషన్ లైసెన్స్ ఒప్పందం (CLA)తో అంగీకరించాలి, అంటే మీరు మీ కాంట్రిబ్యూషన్ ఉపయోగించే హక్కు కలిగి ఉన్నారని, వాస్తవానికి ఇచ్చారని ప్రకటించడం. వివరాలకు, సందర్శించండి <https://cla.opensource.microsoft.com>.

మీరు పుల్ రిక్వెస్ట్ సమర్పించినప్పుడు, CLA బాట్ స్వయంచాలకంగా మీరు CLA సమర్పించాల్సిన అవసరం ఉన్నదో లేదో నిర్ణయించి PR కు సరైన అలంకరణ (ఉదా: స్థితి తనిఖీ, కామెంట్) లభిస్తుంది. బాట్ ఇచ్చే సూచనలను అనుసరించండి. మా CLA ఉపయోగించే అన్ని రిపోస్ లో మీరు కేవలం ఒకసారి మాత్రమే ఇది చేయవలసి ఉంటుంది.

ఈ ప్రాజెక్ట్ [Microsoft Open Source కోడ్ ఆఫ్ కండక్ట్](https://opensource.microsoft.com/codeofconduct/) ను స్వీకరించింది.
అత్యంత సమాచారం కోసం చూడండి [కోడ్ ఆఫ్ కండక్ట్ FAQ](https://opensource.microsoft.com/codeofconduct/faq/) లేదా ఏవైనా ప్రశ్నలు లేదా అభిప్రాయాలు ఉంటే [opencode@microsoft.com](mailto:opencode@microsoft.com) ను సంప్రదించండి.
